# **☁️ミニゲームのREADMEへようこそ☁️**
## 👾 くもぼうやがゲームデビュー！

「くもぼうや」とは、友人との会話中に生まれたオリジナルキャラクターのことです。  
会話の中で頻出する「なんやて」というワードをテーマに、文字をゴールへシュートして遊ぶミニゲームを作りました！

## 🌟 ゲーム概要
- **タイトル**: くもぼうやのひらがな投げミニゲーム
- **ジャンル**: 2Dなんやてアクション
- **プラットフォーム**: Webブラウザ


## 🎮 遊び方
1. 編集中

2. ランキング画面について同一スコアの場合の表示順は、サーバーの処理順に依存するため不規則となります。（システム仕様上の制限）

### 操作方法
- **移動**: A,D または ←,→
- **シュート**: スペースキー(長押しでチャージ)

<details>
<summary>Ver1.1</summary>

【新規実装】
- ゴールを配置。当たり判定を追加。上下に動くように仕様を追加。
- お邪魔棒を配置。当たり判定を追加。棒の中心を軸に回転するよう仕様を追加。

- 一定時間の間、重力が極端に重くなる仕様を追加。10秒ごとに5秒間作動。
- 一定時間の間、ゴールが消えて見える（以降、ゴール隠蔽と呼ぶ）仕様を追加。10秒ごとに5秒間作動。
- 重力・ゴール隠蔽の効果発動が視覚的に分かるよう、演出を実装。
- 重力・ゴール隠蔽の効果発動までをカウントするタイマーを表示。
- ゴールの壁部分に「なんやて」以外の文字を当てると重力・隠蔽タイマーがリセットされる仕組みを実装。リセットは片方ずつ行い、①重力②隠蔽 の順で繰り返す。
- スコア機能のデモ版を実装。
- タイトル画面を追加。現段階ではゲームのプレイ開始のみ選択することが出来る。
- プレイモードの選択画面を追加。降ってくる文字やスコア表示などの文字色を任意の色に変更できる機能を実装。
- 1プレイ99秒とする時間制限を設定。プレイモードの1つとする予定。

【不具合修正・仕様調整】

- くもぼうやが手に持った文字の上に、降ってくる文字が乗り続ける不具合を修正。
- くもぼうやの可動域が広すぎる不具合を修正。
